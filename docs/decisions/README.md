# Architecture Decision Records

Use this folder to capture key technical and product decisions. Follow the ADR template below for each entry:

```
# ADR N: Title
- **Date:** YYYY-MM-DD
- **Status:** Proposed | Accepted | Superseded
- **Context**
- **Decision**
- **Consequences**
```
